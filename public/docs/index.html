    Query Parameters

    sort  optional
    Field to sort by. Defaults to 'id'.

    fields[person]  optional
    Comma-separated fields to include in the response

    filters[first_name]  optional
    Filter by first name.

    filters[last_name]  optional
    Filter by last name.

    include  optional
    Include related information, accepted values 'addresses','emailaddresses','phones','socialmediaaddresses','notes','tags'.
